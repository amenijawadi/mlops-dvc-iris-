# mlops-dvc-iris

Dépôt pédagogique minimal pour TP MLOps consacré au **data versioning** avec **DVC**, utilisant **MinIO** comme remote S3 local (fichier `docker-compose.yml` inclus). 

Le repo fournit le **jeu de données Iris** (`iris.csv`) destinée à illustrer le suivi des données (pointeurs DVC dans Git, blobs en remote) et la **reproductibilité** des artefacts. 

Conçu pour être **forké**.
